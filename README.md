# Cyber Warrior Command Center 2.0

The master hub for the **Cyber Warrior Program** — central landing page for
every CompTIA A+, Network+, Security+, and CySA+ study resource built so far.
This is the successor to the original `Cyber-Warrior-Command-Center` repo,
which now redirects here.

Live site: https://rafikiscyent888.github.io/Cyber-Warrior-Command-Center-2.0/

## What's different from 1.0

Two layers now work together instead of one:

1. **This hub** — one page, all five subject areas, direct links to every
   quiz/sim/acronym site, searchable.
2. **Standalone tile pages** — each subject area also has its own single-tile
   page (Royal Blue background, one colored tile, same links) that can be
   shared on its own without the rest of the hub. Every tile here has a
   "Full Tile ↗" button linking to its standalone counterpart.

| Subject | Standalone tile |
|---|---|
| A+ Core 1 | https://rafikiscyent888.github.io/A-Core-1-tile/ |
| A+ Core 2 | https://rafikiscyent888.github.io/A-Core-2-tile/ |

## What's here

- `index.html` — the command center itself. Plain HTML/CSS/vanilla JS, no
  build step. Color-coded tiles link out to every resource site plus their
  standalone counterpart, with a search bar to filter by keyword.
  Mobile-friendly.
- `CONCEPTS.md` — cross-repo concept index: which topics are already
  covered, in which repos, in what format, and where taxonomies or content
  don't yet line up across repos.
- `CLAUDE.md` — full project context (repo inventory, formats, known content
  issues) for anyone (human or AI assistant) picking this project back up.
